# 亿乐社区 PHP 类
Author: Prk

最全面的亿乐社区 PHP 类！

## 一、目录树

- `yl.users.class.php` 用户接口类
- `yl.goods.class.php` 商品接口类
- `yl.order.class.php` 订单接口类
- `yl.admin.class.php` 后台接口类
- `yl.ghs.class.php` 供货商接口类

因部分原因，我主要维护供货商接口，其它的只是顺便做的。不出意外的情况下 2023 年我将开发出自己的社区系统，即使这样我也会将本仓库在未来做完后归档。
